# Call Center Conversion Analysis

EDA-проект: анализ исходящих звонков клиентам и факторов, влияющих на успешный исход (конверсию).  
Результат — выводы + практические рекомендации для улучшения скрипта и процесса обзвона.

---

## Project overview
**Задача:** понять, какие признаки связаны с успешным исходом звонка (успех/отказ) и какие изменения могут повысить конверсию.  

**Что сделано в проекте:**
- базовая очистка и первичный анализ данных (EDA)
- сравнение успешных и неуспешных звонков
- проверка продуктовых гипотез (длительность, возраст, пол и др.)
- формирование рекомендаций для улучшения конверсии

---

## Repository structure
- **`01_eda_calls.ipynb`** — основной ноутбук с анализом, графиками и выводами  
- **`presentation_calls.pdf`** — краткая презентация результатов и рекомендаций

---

## Key insights
- Доля успешных звонков — примерно **15–20%**
- **Длительность разговора** — самый заметный фактор: успешные звонки в среднем **дольше**
- Основной возрастной сегмент клиентов — **25–45**
- Пол клиента почти **не влияет** на результат (эффект слабый/незначимый)

---

## Recommendations
- Оптимизировать скрипт и обучение операторов так, чтобы удерживать разговор **≥ 60 секунд**
- Для клиентов **до 30 лет** протестировать отдельный скрипт/оффер (другая подача/ценность)
- Проверить гипотезу распределения звонков по сочетанию пола менеджера и клиента (как **A/B тест**)
